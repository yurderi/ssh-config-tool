# ssh-config-tool

> A lightweight ssh-config tool which lets you configure your .ssh/config using YAML

#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build


```
